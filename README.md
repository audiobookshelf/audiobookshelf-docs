# Audiobookshelf Docs Website

Hosted at [audiobookshelf.org](https://audiobookshelf.org/)

Built with [Docusaurus](https://docusaurus.io/). Requires Node.js 20 or later.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

Starts a local development server. Most changes are reflected live without restarting.

## Build

```bash
npm run build
```

Generates static content in the `build` directory.
