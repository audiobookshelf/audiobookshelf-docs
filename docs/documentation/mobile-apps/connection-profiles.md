The Audiobookshelf beta app supports multiple connection profiles (users or servers).

:::info

All downloaded media is connected to the profile used to download it and progress will only be synced when connected to the server using the same profile. This means if you have two separate profiles, one for your internal IP and one for VPN/external access, progress will not sync to the server while using the second connection profile.

:::
