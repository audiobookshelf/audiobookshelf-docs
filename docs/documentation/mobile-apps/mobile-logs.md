The mobile app includes a log view for basic troubleshooting. It is intended for quick debugging and sharing, while more verbose logs are available through debug builds and tools such as ADB.

## Log View

The log page shows recent app activity in a readable format. These logs can be copied and shared to assist with debugging issues.

![Log view](/pages/app_logging/app_logs.png)

## Server Address Redaction

Server addresses are redacted by default to make logs easier to share.

If you need to see the server address that was used, open the options menu and select `Unmask Server Address`.

![Unmask server address](/pages/app_logging/app_logs_mask.png)
