This page describes how to download books from your server onto an Android device and make them available to other apps.

Reasons you may want to do this include:

- Improve your listening experience by reducing lag from poor internet connections
- Limit the impact of audiobook listening on your mobile data usage
- Let other apps on your device access ABS downloads

The ABS app does not support media from other sources. To listen to media in the ABS app, files must be downloaded through the ABS app. Shared storage is only intended to let other apps access ABS media.

## Set Up Shared Storage

1. Open the three-dot menu.
2. Select `Local Media`.
3. Under `Select Media Type`, choose whether you are adding a folder for books or podcasts. For this example, we will be downloading books. Then tap `New Folder`.

![Set up shared storage](/pages/android_app_shared_storage/als_img1.png)

The next few steps vary by device. This example is from a Samsung Z Flip4, but other devices should follow a similar process.

1. Select `Create New Folder`.
2. Choose any location and folder name you want. For this example, the folder is named `Audiobookshelf` and placed at the storage root.
3. Select `Use This Folder`.
4. Allow Audiobookshelf to access the new folder.

![Create folder](/pages/android_app_shared_storage/als_img2.png)

## Download a Book

1. Open any audiobook you want to download. The green button will say `Stream` until the download is complete. Select `Download`.
2. Select the folder you created earlier.
3. You can select `Internal App Storage` if you want, but the files will be stored in the app's default storage location.
4. Press `OK` to confirm the download.

![Download book](/pages/android_app_shared_storage/als_img3.png)

## Finished

When the download is complete, a new icon appears at the top of the app showing how many files are downloading. Tap that icon to view a detailed list of the files being downloaded.

Once the download finishes, the green button changes from `Stream` to `Play`.

![Finished view](/pages/android_app_shared_storage/als_img4.png)
